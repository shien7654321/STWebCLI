# STWebCLITemplate

Web CLI template in the specified branch.

## Vue 3 Project With Webpack

```bash
# install dependencies
yarn

# serve with hot reload at localhost
yarn start

# build for production with minification
yarn build

# code inspection
yarn lint

# style code inspection
yarn lint-style

# code formatter
yarn prettier
```
