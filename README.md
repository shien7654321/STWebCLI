# STWebCLITemplate

Web CLI template in the specified branch.

## Vue 2 Project With Webpack

```bash
# install dependencies
npm install

# build for production with minification
npm run build

# serve with hot reload at localhost
npm run dev

# code inspection
npm run lint

# style code inspection
npm run lint-style

# code formatter
npm run prettier
```
